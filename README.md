### What's wrong with it?

- meta tags missing
- no lang attribute
- no title tag
- more than one h1
- too little semantic markup
- no proper heading hierarchy
- no proper form controls
- no proper image markup
- no proper link markup
- no proper list markup
- inline CSS styles
- email as a link instead of mailto
